Yazılım Mantığı (State Machine)

BMS yazılımı, güvenliği sağlamak için katı bir "Durum Makinesi" (Finite State Machine) mimarisi kullanır. Python kodu, sensörlerden gelen veriyi sürekli izleyerek bu durumlar arasında geçiş yapar.

1

INIT & SELF-CHECK

Sistem açılışta sensörleri ve mosfetleri kontrol eder.

2

PRE-CHARGE

Kapasitörlerin ani akım çekmesini önlemek için akım yavaşça verilir.

3

ACTIVE RUN

Normal çalışma. Voltaj, Akım, Sıcaklık sürekli izlenir.

core_logic.py
class BatteryManager: def __init__(self): self.state = "IDLE" self.max_voltage = 126.0 # 30S Limit def check_safety(self, sensors): # Gerçek zamanlı voltaj kontrolü if sensors.voltage > self.max_voltage: self.trigger_fault("OVER_VOLTAGE") return False # Isı kontrolü (Termal Throttling) if sensors.temp > 60.0: self.limit_current(0.5) # %50 Güç düşümü return True def run_cycle(self): data = self.input_card.read_all() if self.check_safety(data): self.update_mosfets(True) else: self.emergency_shutdown()
