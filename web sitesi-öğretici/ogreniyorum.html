Sıfırdan Web Sitesi Yapımı

Adım Adım Kendi Projeni Geliştir

Bu rehberde, hiç tecrüben olmasa bile kendi web siteni nasıl yapacağını adım adım öğreneceksin. Web geliştirme, üç temel teknolojinin birleşimidir: HTML, CSS ve JavaScript. Bu rehberde, bu üçünü kullanarak çalışan, basit bir proje yapacağız. Her adımı dikkatlice oku ve kendi bilgisayarında uygula. Hazırsan, başlayalım!

Bölüm 1: Hazırlık ve Doğru Araçlar

Bir işe başlamadan önce alet çantasına ihtiyacımız var. Bizim aletlerimiz yazılımlar olacak.

Adım 1.1: Kod Editörü (Bizim Atölyemiz)

Yazdığımız kodları renklendiren, hatalarımızı gösteren ve işimizi kolaylaştıran programlara "kod editörü" denir. Not defteri bile kullanılabilir, ancak bu bir at arabasıyla yarışa girmeye benzer.

Bizim Seçimimiz: Visual Studio Code (VS Code)

VS Code, ücretsiz, hızlı ve dünyadaki geliştiricilerin %80'inden fazlasının kullandığı standart editördür. Onu hemen resmi sitesinden indirip kurmalısın.

Süper Güç Eklentisi: Live Server

VS Code'u kurduktan sonra, soldaki "Extensions" (Kare ikon) sekmesine tıkla, arama kutusuna "Live Server" yaz ve ilk çıkanı yükle. Bu eklenti sayesinde, HTML dosyanı her kaydettiğinde tarayıcın otomatik olarak yenilenecek. Bu bize inanılmaz zaman kazandıracak.

Adım 1.2: Web Tarayıcısı (Bizim Sahnemiz)

Kodlarımızın son kullanıcıya nasıl göründüğünü test edeceğimiz yerdir. Muhtemelen zaten bir tane kullanıyorsun (Google Chrome, Firefox, Safari, Edge vb.).

Tavsiyemiz: Google Chrome veya Firefox.

Bu tarayıcılar, "Geliştirici Araçları" (DevTools) konusunda en iyileridir. Sayfandayken F12 tuşuna (veya sağ tıklayıp 'İncele') basarak bu araçları açabilirsin. Şimdilik sadece nasıl açıldığını bilmen yeterli. Hata ayıklamak için burayı çok kullanacağız.

Adım 1.3: Proje Klasörünü Oluşturma (Bizim Planımız)

Düzen, profesyonelliğin ilk adımıdır. Masaüstünde veya "Belgelerim" içinde **"ilk-sitem"** adında bir klasör oluştur. Tüm dosyalarımızı bu klasörün içinde tutacağız.

Şimdi VS Code'u aç, "File" > "Open Folder..." (Dosya > Klasör Aç) menüsünden bu yeni oluşturduğun **"ilk-sitem"** klasörünü seç. Artık çalışma alanımız hazır.

VS Code'un solundaki "Explorer" panelinde, "ilk-sitem" klasörünün üzerine gel ve şu üç dosyayı tek tek oluştur (Yeni Dosya ikonuna tıklayarak):

  • index.html
  • style.css
  • script.js

Bu İsimler Neden Önemli?

  • index.html: Sunucular, bir siteye girildiğinde varsayılan olarak bu isimdeki dosyayı arar. Ana sayfanız *her zaman* index.html olmalıdır.
  • .html, .css, .js: Bunlar dosya uzantılarıdır. Bilgisayara bu dosyanın bir metin değil, bir HTML sayfası, bir CSS stil dosyası veya bir JavaScript kodu olduğunu söylerler.

Bölüm 2: HTML ile Sayfa İskeletini Oluşturma

HTML (HyperText Markup Language), sayfamızın iskeletidir. Başlıkları, paragrafları, resimleri ve linkleri HTML "etiketleri" ile belirleriz.

Adım 2.1: HTML İskeleti (Boilerplate)

index.html dosyana tıkla. Boş dosyaya sadece bir ünlem işareti (!) yaz ve ardından Enter veya Tab tuşuna bas.

VS Code senin için sihirli bir şekilde tüm temel HTML iskeletini oluşturacaktır:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

</body>
</html>

Bu Etiketler Ne Anlama Geliyor?

  • <!DOCTYPE html>: Tarayıcıya "Bu modern bir HTML5 sayfasıdır" der.
  • <html lang="en">: Sayfanın ana dilinin İngilizce olduğunu söyler. Bunu lang="tr" olarak değiştirelim.
  • <head>: Sayfanın görünmeyen ayar kısmıdır (Başlık, CSS dosyaları, meta veriler).
  • <title>Document</title>: Tarayıcı sekmesinde görünen başlıktır. Bunu <title>Benim İlk Sitem</title> olarak değiştirelim.
  • <body>: Sayfada gördüğün *her şey* (yazılar, resimler, butonlar) bu etiketin içine yazılır. Bizim tüm işimiz burayla.

Adım 2.2: Sayfaya İçerik Ekleme

Şimdi <body> etiketinin içini, sitemizin içeriğiyle dolduralım. Aşağıdaki kodları kopyalayıp <body> ve </body> etiketlerinin arasına yapıştır.

    <h1 id="ana-baslik">Merhaba Dünya!</h1>
    <p>Bu benim web geliştirmeyi öğrenirken yaptığım ilk site.</p>

    <h2>Öğrendiklerim</h2>
    <ul>
        <li>HTML: Sayfa iskeleti</li>
        <li>CSS: Sayfa stili</li>
        <li>JavaScript: Sayfa interaktifliği</li>
    </ul>

    <h2>Harekete Geç!</h2>
    <button id="mesaj-butonu">Bana Tıkla!</button>

Adım 2.3: Sayfayı Tarayıcıda Açma (Live Server)

Dosyanı kaydet (Ctrl+S veya Cmd+S). Şimdi index.html dosyanın üzerinde sağ tıkla ve **"Open with Live Server"** seçeneğine tıkla.

Tebrikler! Tarayıcın açıldı ve ilk web sayfan karşında. Henüz çirkin görünüyor, ama çalışıyor. Şimdi onu güzelleştirme zamanı.

Bölüm 3: CSS ile Sayfayı Şekillendirme

CSS (Cascading Style Sheets), iskeletimizi giydirmemizi sağlar. Renkler, yazı tipleri, hizalama, boşluklar... Hepsi CSS ile kontrol edilir.

Adım 3.1: CSS Dosyasını HTML'e Bağlamak

CSS kodlarını doğrudan HTML içine yazabilirdik (inline style) ama bu çok dağınık ve kötü bir yöntemdir. Bunun yerine tüm stillerimizi style.css dosyasında tutacağız.

Peki HTML, bu CSS dosyasını kullanacağını nereden bilecek? Biz söyleyeceğiz. index.html dosyana geri dön ve aşağıdaki <link> etiketini <head> bölümünün içine (<title> etiketinin hemen altına) ekle.

<head>
    ...
    <title>Benim İlk Sitem</title>
    <link rel="stylesheet" href="style.css"> <!-- BU SATIRI EKLE -->
</head>

Bu satır, tarayıcıya "Bu bir stil dosyasıdır, adı da style.css'tir. Git ve onu yükle." der.

Adım 3.2: İlk CSS Kuralımız (Seçici, Özellik, Değer)

Şimdi style.css dosyasına tıkla. Bir CSS kuralı 3 bölümden oluşur:
secici { ozellik: deger; }

Sayfamızın tamamını etkileyerek başlayalım. body etiketini seçelim ve ona bir arkaplan rengi ile daha okunaklı bir yazı tipi verelim.

/* Sayfanın tamamına etki eden genel stiller */
body {
    background-color: #f4f4f4; /* Çok açık bir gri arkaplan */
    font-family: Arial, sans-serif; /* Modern, tırnaksız bir yazı tipi */
    line-height: 1.6; /* Satır aralığını artırarak okunabilirliği sağlar */
    padding: 20px; /* İçeriği ekranın kenarlarından uzaklaştırır */
}

Dosyayı kaydettiğin anda (Ctrl+S), Live Server sayesinde tarayıcının anında değiştiğini göreceksin. Sihir gibi!

Adım 3.3: Sayfayı Güzelleştirelim

Şimdi diğer elemanlara da stil verelim. Aşağıdaki kodları style.css dosyana (body kuralının altına) ekle.

/* Sadece h1 ve h2 başlıklarını seçer */
h1, h2 {
    color: #333; /* Koyu gri bir başlık rengi */
}

/* Tüm liste elemanlarının başındaki noktayı kaldırır */
ul {
    list-style-type: none; /* Madde imlerini kaldırır */
    padding: 0;
}

/* Buton için özel stil */
button {
    background-color: #007acc; /* Güzel bir mavi arkaplan */
    color: white; /* Beyaz yazı rengi */
    padding: 10px 15px; /* İç boşluk (butonu büyütür) */
    border: none; /* Çirkin varsayılan kenarlığı kaldırır */
    border-radius: 5px; /* Kenarları yuvarlatır */
    cursor: pointer; /* Üzerine gelince fare imlecini el yapar */
}

/* Butonun üzerine gelince (hover) rengini biraz değiştir */
button:hover {
    background-color: #005f99;
}

Kaydet ve tarayıcına bak. Siten artık çok daha profesyonel görünüyor! Sadece bir eksiğimiz kaldı: Buton çalışmıyor.

Bölüm 4: JavaScript ile İnteraktiflik

JavaScript (JS), sayfamıza hayat verir. Kullanıcı bir butona tıkladığında ne olacağını, bir form doldurulduğunda ne yapılacağını JS ile belirleriz. O, sitemizin "beynidir".

Adım 4.1: JavaScript Dosyasını Bağlamak

Tıpkı CSS gibi, JS kodunu da ayrı bir dosyada tutacağız. Ve tıpkı CSS gibi, onu da HTML'e bağlamamız gerekiyor.

ÇOK ÖNEMLİ: <script> etiketi, CSS'in aksine <head> içine *değil*, <body> etiketinin **kapanışından hemen önceye** (en alta) konur.

Neden? Çünkü JS, HTML elementlerini (buton, başlık vb.) manipüle edecektir. Tarayıcının, JS kodunu okumadan önce o HTML elementlerini ekrana çizmiş olduğundan emin olmak isteriz. Eğer script'i en üste koyarsak, JS çalışır, butonu arar ama buton henüz çizilmediği için hata verir.

index.html dosyanın en altına git ve şu satırı ekle:

    ...
    <button id="mesaj-butonu">Bana Tıkla!</button>

    <script src="script.js"></script> <!-- BU SATIRI EKLE -->
</body>
</html>

Adım 4.2: İnteraktifliğin 3 Adımı (Seç, Dinle, Değiştir)

Yapacağımız şey basit: Butona tıklandığında, <h1> başlığının metnini değiştireceğiz. JavaScript'te bu her zaman 3 adımlı bir süreçtir:

  1. Seç: Değiştireceğimiz veya dinleyeceğimiz HTML elementlerini seç.
  2. Dinle: Bir olayın (örn: 'tıklama') gerçekleşmesini bekle.
  3. Değiştir: Olay gerçekleştiğinde, bir eylem yap (örn: metni değiştir).

Şimdi script.js dosyana tıkla ve içine aşağıdaki kodları yaz:

// 1. ADIM: HTML'deki elementleri ID'leri ile YAKALA (SEÇ)
// "document" tüm HTML sayfasını temsil eder.
// getElementById metodu, verdiğimiz ID'ye sahip elementi bulur.

const mesajButonu = document.getElementById('mesaj-butonu');
const anaBaslik = document.getElementById('ana-baslik');


// 2. ADIM: Butona bir 'tıklama' dinleyicisi ekle (DİNLE)
// 'click' olayı gerçekleştiğinde, function() içindeki kodları çalıştır.

mesajButonu.addEventListener('click', function() {

    // 3. ADIM: Olay gerçekleştiğinde, başlığın metnini DEĞİŞTİR
    anaBaslik.textContent = 'Harika! JavaScript öğreniyorsun!';

    // Bonus: Başlığın rengini de değiştirelim
    anaBaslik.style.color = '#007acc';
});

Dosyayı kaydet. Şimdi tarayıcına git ve butona tıkla. Başlığın sihirli bir şekilde değiştiğini göreceksin!

Tebrikler ve Sonraki Adımlar

Harika bir iş çıkardın! Şu an elinde HTML (iskelet), CSS (stil) ve JavaScript (beyin) ile çalışan, tam fonksiyonlu bir mini web projesi var. Web geliştirmenin temellerini attın.

Bu sadece bir başlangıç. Kendini daha da geliştirmek için sitemizdeki diğer detaylı bölümleri incelemeye başlayabilirsin:

  • HTML: Daha fazla etiket öğren (resimler <img>, tablolar <table>, formlar <form>...).
  • CSS: Daha karmaşık tasarımlar yap (Flexbox, Grid, animasyonlar...).
  • JavaScript: Daha gelişmiş özellikler ekle (form kontrolü, veri alma...).

Unutma, en iyi öğrenme yolu sürekli pratik yapmak ve bozmaktan korkmamaktır. Bol bol kod yaz!
