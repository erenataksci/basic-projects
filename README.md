# 📂 Basic Projects & Mini Software Utilities

Welcome to my central repository for foundational full-stack web applications, automation scripts, and localized database management utilities. This space acts as a structured archive for practical implementation patterns across Python, JavaScript, and database integration.

---

## 🛠️ Repository Structure & Project Breakdown

### 🤖 1. akillibms
* **Core Technology:** JavaScript / Node.js & HTML5 Ecosystem
* **Description:** An algorithmic simulation or control logic framework designed to interact with Battery Management Systems (BMS). It maps cell behaviors, state-of-charge tracking logic, and telemetry data visualization.

### 🌦️ 2. havadurumu
* **Core Technology:** Python 3.x & JSON Data Pipelines
* **Description:** A local weather tracking utility integrated with regional location definitions (`il-ilce.json`). It processes multi-city queries, parses live meteorological data structures, and outputs filtered forecasts.

### 🔐 3. şifrelifirebasekayitgirisnotsistemi
* **Core Technology:** Firebase Auth, Firestore & Vanilla JS / Node.js
* **Description:** A secured multi-user authentication system featuring encrypted session states, user registration pipelines, and a persistent cloud-backed note-taking engine built over Firebase.

### 📝 4. şifreşinotdefteri
* **Core Technology:** Client-Side Cryptography & Web Storage API
* **Description:** A lightweight, local-first secure notepad application. It focuses on standalone cryptographic workflows to lock, unlock, and manage text assets directly within individual session instances.

### 🌐 5. imwebsite
* **Core Technology:** HTML5, Modern CSS3 & Modular JavaScript
* **Description:** My production-grade personal hub and professional engineering portfolio showcase. It contains dedicated technical sub-pages highlighting core R&D milestones, including:
  * Alternative energy frameworks (Hydrogen & Thermal systems)
  * Hardware/Software integration pipelines (Jarvis Assistant nodes)
  * Advanced physical science publications and engineering portfolios

### 📖 6. web sitesi-öğretici
* **Core Technology:** Structural HTML5 & Responsive CSS Layouts
* **Description:** An educational, modular web repository tracking modern front-end layout paradigms. It serves as an open template highlighting vanilla styling architecture, responsive flexbox models, and grid mechanics.
